# Resume Checklist

When restoring context:

1. Read the newest relevant handoff.
2. Check `git status --short`.
3. Verify whether referenced files still exist.
4. Re-run only cheap status commands first.
5. Identify what changed since the handoff.
6. Restate current goal and next action before editing.

## Do Not

- Assume the handoff is current.
- Re-run expensive benchmarks without user intent.
- Revert work not mentioned in the handoff.
- Treat uncertain notes as facts.
